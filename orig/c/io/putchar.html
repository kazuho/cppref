putchar

Syntax: 

    #include <cstdio>
    int putchar( int ch );

The putchar() function writes ch to stdout. The code 

     putchar( ch );

is the same as 

     putc( ch, stdout );

The return value of putchar() is the written character, or EOF if there is an error.
